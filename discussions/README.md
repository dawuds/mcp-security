## Contributing Discussions

Have a topic that deserves deeper exploration? Open an issue or submit a PR with a new topic file in `discussions/topics/`. Good discussion topics are:

- Specific to MCP security (not general AI/LLM security)
- Not already covered adequately by the existing docs
- Grounded in concrete attack scenarios or real deployment challenges
- Balanced — covering both the risk and reasonable mitigations
